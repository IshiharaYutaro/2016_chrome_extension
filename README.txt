01.browser_action_faebook_share
facebookのapiを利用し、jsonで情報を取得、
popupでfacebookシェア数を表示する。
02.browser_action_hatena_share
hatenaのapiを利用し、jsonで情報を取得、
popupでhatenaブックマーク数を表示する。
03.browser_action_weather
livedoorの気象情報のjsonを取得し、
東京の明日の天気をpopup表示
04.browser_action_weather_image
気象庁のレーダー画像(10分前)を取得し、
popup表示する。
05.chorome_icon_cheak
Googleの検索画面にfaviconを表示する。
06.chrome_inline_cheak
Googleの検索画面に検索結果と似ているキーワードを
ライン表示する。
07.clipboard_history
メモを表示し、削除ボタンで消すことができる。
右クリックから保存可能
08.logo_change
google検索のgooleのロゴ表示をYahooロゴの変更する。
09.new_tab_logo
新しいtabを表示した際に、Yahooロゴを表示する。
10.nihonyuusei_search
日本郵政の伝票を検索する。
右クリックから検索と
popup表示から伝票検索可能
11.post_facebook

ボタンを押した場合は、

13.post_twitter

選択した場合その内容を投稿。

15.search_year

表示する様に変更する。
16.tab_change
現在、開いているタブ、windowをすべて表示する。
